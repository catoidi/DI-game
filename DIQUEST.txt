>>Command Structure:
>>Arg 1: Which map to spawn the debuff/objective in. "*" will spawn in every map.
>>Arg 2: Specify either "Debuff" or "Objective" here.
>>Arg 3: Class name of your chosen debuff or objective.
>>Arg 4: (Optional) Chance of quest/debuff being added to the NPC. Default/Maximum is 256, which is guaranteed to appear.
>>Arg 5: (Optional) Minimum wave required to add this debuff/objective. Default is 0, which is every wave.
